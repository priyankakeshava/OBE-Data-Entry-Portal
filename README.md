# OBE Data Entry Portal
https://priyankakeshava.github.io/OBE-Data-Entry-Portal/
A web-based Outcome Based Education (OBE) portal for managing:

- Course Details
- Course Outcomes (CO)
- Program Outcomes (PO)
- Program Specific Outcomes (PSO)
- Outcome Mapping
- Validation Forms
